## Présentation
Relax Shutter est une plateforme de médias sociaux dédiée au partage d'expériences de week-end et de voyage à travers la photographie. C'est un espace où les utilisateurs peuvent documenter leurs aventures, découvrir de nouvelles destinations et créer des liens avec d'autres voyageurs passionnés.


## Technologies utilisées

- **Laravel**:  Framework PHP backend pour API et authentification.
- **VueJS**:  Framework frontend pour interfaces utilisateur réactives.
- **TailwindCSS**: Framework CSS utility-first pour le style.
- **MySQL**: Base de données relationnelle pour le stockage.
