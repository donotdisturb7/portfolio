## Niveau 2 - Partir des exigences et aller jusqu'à une application complète

### Élaborer et implémenter les spécifications fonctionnelles et non fonctionnelles à partir des exigences

**Description :** Transformer les besoins métier en spécifications techniques détaillées et les implémenter.

**Ce que j'ai fait dans le Site E-commerce :**
- Analysé le cahier des charges client pour identifier toutes les exigences
- Défini les spécifications fonctionnelles : catalogue, panier, paiement, gestion commandes
- Établi les spécifications non-fonctionnelles : performance (moins de 3s), sécurité, disponibilité
- Documenté l'architecture technique complète avec Vue.js et APIs
- Implémenté toutes les fonctionnalités selon les spécifications validées

**Technologies utilisées :** Analyse des besoins, documentation technique, Vue.js, architecture logicielle

### Appliquer des principes d'accessibilité et d'ergonomie

**Description :** Concevoir des interfaces utilisables par tous en respectant les standards d'accessibilité.

**Ce que j'ai fait dans le Site E-commerce :**
- Respecté les standards WCAG 2.1 avec contrastes de couleurs appropriés
- Ajouté des labels explicites et aria-labels sur tous les éléments interactifs
- Implémenté la navigation au clavier pour toutes les fonctionnalités
- Créé des alternatives textuelles pour toutes les images et icônes
- Testé l'accessibilité avec des lecteurs d'écran et corrigé les problèmes

**Technologies utilisées :** Design responsive, 

### Adopter des bonnes pratiques de conception et de programmation

**Description :** Structurer le code de manière maintenable, évolutive et conforme aux standards.

**Ce que j'ai fait dans le Site E-commerce :**
- Organisé le code en composants Vue.js réutilisables et modulaires
- Appliqué les principes SOLID et DRY pour éviter la duplication
- Utilisé Vuex pour la gestion d'état centralisée et prévisible
- Commenté le code et créé une documentation technique complète
- Mis en place un système de gestion des erreurs cohérent

**Technologies utilisées :** Vue.js, composants, routes
