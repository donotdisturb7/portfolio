### Identifier les acteurs et les différentes phases d'un cycle de développement

**Description :** Cartographier les parties prenantes et structurer le projet selon une méthodologie de développement.

**Ce que j'ai fait dans le Stage API Python :**
- Identifié tous les stakeholders : équipes métier, architectes, développeurs, testeurs, ops
- Défini les rôles : tech lead, développeurs senior/junior, testeur QA, DevOps engineer
- Structuré le projet en sprints de 2 semaines avec livrables incrementaux
- Planifié les phases : analyse des besoins, conception API, développement, tests, déploiement
- Organisé les cérémonies : daily standups, sprint planning, code reviews, rétrospectives

**Technologies utilisées :** définition des rôles d'équipe

### Formaliser les besoins du client et de l'utilisateur

**Description :** Transformer les besoins exprimés en spécifications techniques détaillées et documentées.
